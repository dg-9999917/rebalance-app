# 비율 기반 리밸런싱 추적 웹앱

6/12 원금을 기준으로 "승자 몰아주기" 방식의 포트폴리오 리밸런싱을 추적하는 단일 HTML 웹앱.

## 실행 방법

`index.html` 파일을 더블클릭하면 브라우저에서 바로 열립니다.  
별도 서버·빌드 도구 없이 동작합니다.

## 핵심 개념

- **현재비중 합 > 100** = 수익 (분모가 6/12 원금 1%로 고정)
- **설정비율**: 목표 비중. 합 = 100.
- **조정비율**: 리밸런싱 시 새로 바꾸는 비중. 줄인 종목 매도 → 늘린 종목 비율 비례 매수.
- 데이터는 localStorage 자동 저장. 백업/불러오기/초기화 버튼 제공.

## 파일 구조

```
index.html   — 앱 전체 (HTML + CSS + JS 인라인)
PROGRESS.md  — 구현 진행 상황
README.md    — 이 파일
```
