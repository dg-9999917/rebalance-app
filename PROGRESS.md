- [x] **검증 (가)~(바) 통과**
  — (가) 주식 adjR=8 → 현금 adjR=92
  — (나) adjR 합=100 → 확정 저장 버튼 활성화
  — (다) 주식 adjR 불변, (라) r 불변, (마) 현금 q 불변
  — (바) 주식 adjR>100 → 현금 adjR=0 + 안내
